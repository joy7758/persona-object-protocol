# POP v0.1.5 Plan

## Release Intent

`v0.1.5` is a LangChain execution contract release.

## Scope

- stabilize the LangChain execution contract surface rather than
  primarily expanding helper count
- keep `create_langchain_execution_bundle(...)` as the main public
  LangChain entry point
- validate `create_langchain_create_agent_kwargs(...)`,
  `create_langchain_context_bundle(...)`, and
  `create_langchain_middleware_bundle(...)` as layered support surfaces
- add installed-package and optional-dependency smoke validation for the
  LangChain contract path

## Secondary Scope

- CrewAI remains secondary and compatibility-focused in this release
  line
- CrewAI helpers remain early preview and runtime-facing

## Out of Scope

- no new schema baseline
- no change to the canonical top-level persona boundary
- no official LangChain integration claim
- no full runtime execution requirement

## Compatibility Position

The canonical schema baseline remains `v0.1.0`.
`create_langchain_execution_bundle(...)` is the main public LangChain
entry point in this release line. Compatibility helpers remain
available for backward compatibility, but they are secondary.

## Exit Criteria

- LangChain execution contract semantics are covered by tests
- installed-package and optional-dependency smoke paths are green
- examples remain runnable without live model calls
- release notes prepared
