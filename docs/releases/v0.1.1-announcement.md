# POP v0.1.1 Announcement

POP is now publicly installable and strict-schema-validatable from real
PyPI.

Package release `0.1.1` stabilizes the installable distribution path
and schema-aware validation surface. The canonical schema baseline
remains `v0.1.0`, and this release does not introduce a new structural
persona object boundary.

This release provides:

- an installable Python package
- optional strict JSON Schema validation
- versioned schema visibility from installed package resources
- a CLI inspection path
- public PyPI distribution

POP remains an early preview project. Runtime adapter work continues
next.
