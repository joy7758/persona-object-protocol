# POP v0.1.6 Plan

## Release Intent

`v0.1.6` is a LangChain adoption-friction reduction release.

It should reduce adoption cost for the existing LangChain execution
contract surface rather than expand helper count or alter the protocol
boundary.

## Primary Focus

- installed-package examples for the LangChain contract surface
- a clearer minimal integration path for external users
- improved documentation around the main entry point
  `create_langchain_execution_bundle(...)`
- clearer backward-compatibility guidance for retained helper wrappers
- friction reduction for optional-dependency installation and smoke
  validation

## Out of Scope

- no new schema baseline
- no change to the canonical top-level persona boundary
- no new helper-surface proliferation as a primary goal
- no shift of the release line away from LangChain as the main
  execution target
- no attempt to make CrewAI a parallel primary line in this increment

## Compatibility Position

The canonical schema baseline remains `v0.1.0`. The package may advance
while the schema baseline remains fixed unless an explicit structural
protocol change is introduced later.

CrewAI remains secondary and compatibility-focused in this release line.

## Exit Criteria

- installed-package examples are clear and reproducible
- minimal LangChain integration guidance is easier to adopt
- backward-compatibility position is documented cleanly
- optional-dependency smoke remains green
- no change to the canonical schema baseline
