# POP v0.1.3 Plan

## Release Intent

`v0.1.3` is an execution-layer release.

## Scope

- LangChain `create_agent` / middleware / context helpers
- CrewAI `Agent(...)` initialization helpers
- dependency-light smoke tests
- execution-layer examples and documentation

## Out of Scope

- no new schema baseline
- no change to the canonical top-level persona boundary
- no real model execution requirement

## Compatibility Position

The canonical schema baseline remains `v0.1.0`. `v0.1.3` should extend
runtime-facing helper surfaces without redefining POP core semantics.

## Exit Criteria

- execution-layer helpers implemented
- smoke tests green
- examples runnable without network access
- release notes prepared
