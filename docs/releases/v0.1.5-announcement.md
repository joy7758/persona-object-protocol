# POP v0.1.5 Announcement

POP v0.1.5 is now publicly released on PyPI.

This release stabilizes the LangChain execution contract surface around
`create_langchain_execution_bundle(...)` while preserving the canonical
schema baseline at `v0.1.0`.

The current release provides:

- a publicly installable Python package
- optional strict JSON Schema validation through the `schema` extra
- a LangChain-facing execution contract surface centered on
  `create_langchain_execution_bundle(...)`
- installed-package and optional-dependency validation paths for the
  LangChain contract surface

The package release is `0.1.5`. The canonical schema baseline remains
`v0.1.0`.

This remains an early preview protocol package. The LangChain execution
surface should be understood as a runtime-facing contract layer, not as
an official LangChain runtime integration or a behavior-equivalence
claim.
