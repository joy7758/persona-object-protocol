# POP v0.1.1

## Summary

`v0.1.1` is a packaging and installability fix release for POP.

## Fixed in This Release

- bundled canonical schema resources into the installable package
- updated schema loading so installed distributions can read schema
  resources without relying on source-tree paths
- restored installed-package visibility of available schema versions
- clarified the strict-smoke installation path for the optional schema
  dependency

## Validation Status

- `unittest` passes
- build and `twine check` pass
- installed-package schema visibility is verified as a TestPyPI smoke
  target

## Notes

- the canonical schema baseline remains `v0.1.0`
- this release does not introduce a new schema baseline
- TestPyPI smoke for installed-package schema visibility is the primary
  target of this patch release
