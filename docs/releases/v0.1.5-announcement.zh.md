# POP v0.1.5 发布说明

POP v0.1.5 已正式在 PyPI 上公开发布。

这一版本以 `create_langchain_execution_bundle(...)` 为主入口，稳定了
面向 LangChain 的 execution contract surface，同时保持 canonical
schema baseline 仍为 `v0.1.0`。

当前版本已经具备：

- 可从 PyPI 直接安装的 Python 包
- 通过可选 `schema` extra 执行严格 JSON Schema 校验的能力
- 以 `create_langchain_execution_bundle(...)` 为中心的
  LangChain-facing execution contract surface
- 面向 LangChain contract surface 的安装态与 optional-dependency
  验证路径

当前 package release 为 `0.1.5`，而 canonical schema baseline
仍保持为 `v0.1.0`。

需要说明的是，POP 目前仍处于 early preview 阶段。当前 LangChain
execution surface 的定位是 runtime-facing contract layer，而不是
官方 LangChain runtime integration，也不意味着行为等价保证。
