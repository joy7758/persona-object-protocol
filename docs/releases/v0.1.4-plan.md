# POP v0.1.4 Plan

## Release Intent

`v0.1.4` is a LangChain optional-dependency execution release.

## Scope

- LangChain `create_agent` helper surfaces
- LangChain middleware-facing helper bundles
- LangChain runtime-context helper scaffolds
- dependency-aware smoke tests for optional LangChain integration

## Secondary Scope

- CrewAI remains secondary in this release line
- CrewAI work in this increment is compatibility-focused rather than
  execution-layer-deepening

## Out of Scope

- no new schema baseline
- no change to the canonical top-level persona boundary
- no official runtime integration claim
- no real model execution requirement

## Compatibility Position

The canonical schema baseline remains `v0.1.0`. `v0.1.4` should extend
LangChain-facing execution helpers without redefining POP core
semantics.

Earlier LangChain execution helpers are retained as backward-compatibility
wrappers. The primary release surface should be documented through the
new `create_agent`, middleware-bundle, runtime-context, and execution-bundle
helper names rather than through legacy scaffold names.

## Exit Criteria

- LangChain execution helpers aligned with `create_agent`,
  middleware, and runtime-context helper surfaces
- dependency-aware smoke tests green
- examples runnable without network access
- release notes prepared
