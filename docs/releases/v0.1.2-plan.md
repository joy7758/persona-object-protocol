# POP v0.1.2 Plan

## Release Intent

`v0.1.2` is planned as a runtime adapter release. Its purpose is to
introduce runtime adapter scaffolding and runtime-facing binding
surfaces while keeping the canonical persona schema stable.

## In Scope

- LangChain adapter class scaffold
- CrewAI adapter class scaffold
- adapter contract tests
- runtime adapter documentation
- backward-compatible projection alignment

## Out of Scope

- no new schema baseline
- no change to the canonical top-level persona object boundary
- no full runtime dependency integration yet

## Compatibility Position

The package version may advance as adapter work lands. The canonical
schema baseline remains `v0.1.0` unless an explicit structural protocol
change is introduced in a later release.

## Exit Criteria

- adapter classes merged
- tests green
- documentation updated
- release notes prepared
